# Akira Lampa

Akira is a Lampa UI plugin focused on a cinematic home screen, a compact Apple TV style top bar, TMDB-powered rows, card logos, and a Netflix-like detail view.

## Files

- `akira.js` - current production plugin build.

## Development

This repository intentionally keeps donor/reference plugins out of version control. Temporary experiments and references should stay in `temp/`.
